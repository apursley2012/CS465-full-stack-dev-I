## Step-by-Step Installation & Setup

### Step 1: Download the Project Files

You have two ways to get the project files:

#### Option 1: Download ZIP

- Go to the GitHub repository: [https://github.com/apursley2012/CS465-full-stack-dev-I](https://github.com/apursley2012/CS465-full-stack-dev-I)  
- Click the green **Code** button  
- Select **Download ZIP**  
- Extract/unzip the downloaded file to a folder on your computer, e.g., your Desktop or Documents

#### Option 2: Clone Using Git (Recommended if you plan to continue development)

- Open your terminal or command prompt  
- Navigate to the folder where you want the project (e.g., Desktop):

    ```bash
    cd Desktop
    ```

- Run the clone command:

    ```bash
    git clone https://github.com/apursley2012/CS465-full-stack-dev-I.git
    ```

- Change directory into Module 1’s folder:

    ```bash
    cd CS465-full-stack-dev-I/Module\ 1/travlr
    ```

### Step 2: Review Files

- You should see `.html`, `.css`, `images/`, and other files in this folder.

## How to View the Website

### Method 1: Open Locally in Browser (Simplest)

- Open the folder where you saved/cloned the project  
- Double-click `index.html` to open it in your default web browser  
- Use the navigation bar inside the page to visit other pages (Travel, Meals, Rooms, etc.)

### Method 2: Use a Local Server with Node.js (Optional)

- This method helps when you move to dynamic modules later.

1. Open terminal/command prompt  
2. Make sure you’re in the `travlr` folder (see Step 1)  
3. Install dependencies (if `package.json` exists):

    ```bash
    npm install
    ```

4. Start the server:

    ```bash
    npm start
    ```

5. Open your browser and go to:

    ```
    http://localhost:3000/
    ```
